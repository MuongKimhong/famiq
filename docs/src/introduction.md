## Installation
```
cargo add famiq
```
or adding this to `Cargo.toml`
```toml
[dependencies]
famiq = "0.3.0"
```
Famiq supports only bevy` 0.16.x` onward.

see [crateio](https://crates.io/crates/famiq) & [rustdoc](https://docs.rs/famiq/latest/famiq/).
